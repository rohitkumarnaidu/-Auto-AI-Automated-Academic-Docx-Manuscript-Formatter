# 001 - Python-First Runtime

Status: Accepted

Context: ScholarForm AI relies on Python-first ML and document-processing libraries, and the existing backend codebase is fully Python.

Decision: Standardize on Python 3.12 for backend runtime and CI.

Consequences: Backend tooling, dependencies, and CI workflows are aligned to Python 3.12, and non-Python services are integrated via APIs rather than replatforming.
