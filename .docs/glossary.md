# Glossary

| Term | Definition |
|------|------------|
| **ADR** | Architecture Decision Record — immutable record of a technical decision |
| **Alembic** | Database migration tool for SQLAlchemy |
| **Celery** | Distributed task queue for asynchronous background processing |
| **ChromaDB** | Open-source vector database for RAG embeddings |
| **ClamAV** | Open-source antivirus engine for file upload scanning |
| **CSL** | Citation Style Language — XML-based format for citation formatting |
| **Docling** | IBM document understanding library for PDF parsing |
| **FastAPI** | Python web framework used as the sole API gateway |
| **GROBID** | GeneRation Of BIbliographic Data — ML-based PDF parser (optional, Docker) |
| **Groq** | LLM provider used as secondary fallback in the 3-tier LLM strategy |
| **Jinja2** | Templating engine used for document formatting |
| **LiteLLM** | Library for LLM API routing with provider fallback |
| **Locust** | Python load testing framework |
| **Next.js** | React framework used for the frontend (version 16, App Router) |
| **NVIDIA NIM** | NVIDIA Inference Microservice — primary LLM provider |
| **Ollama** | Local LLM deployment — tertiary fallback in the LLM strategy |
| **OpenCode** | CLI tool for AI-assisted software engineering |
| **Playwright** | Microsoft browser automation framework for E2E testing |
| **Prometheus** | Open-source monitoring and alerting toolkit |
| **RAG** | Retrieval-Augmented Generation — combining vector retrieval with LLM generation |
| **RLS** | Row-Level Security — Supabase PostgreSQL access control |
| **Render** | Cloud platform used for backend hosting |
| **SSE** | Server-Sent Events — HTTP-based real-time communication |
| **Supabase** | Firebase alternative providing PostgreSQL, auth, and storage |
| **Vercel** | Cloud platform used for frontend hosting |
| **Vitest** | Vite-native unit test framework (used in place of Jest) |
| **Zod** | TypeScript-first schema validation library |
