# ScholarForm AI Brand Assets

This directory contains ScholarForm AI logos and brand assets.

## Logo Variants

| File | Description |
|------|-------------|
| `scholarform-logo.svg` | Full logo (wordmark + icon) |
| `scholarform-icon.svg` | Icon only |
| `scholarform-logo-dark.svg` | Dark background variant |
| `scholarform-logo-light.svg` | Light background variant |

## Usage

- **Minimum size:** 32px (icon), 120px (full logo)
- **Clear space:** 25% of logo height on all sides
- **Do not:** rotate, recolor, outline, or add effects to the logo

## Color

- Primary: `#2563EB` (Blue 600)
- Dark: `#1E293B` (Slate 800)
- Light: `#F8FAFC` (Slate 50)
