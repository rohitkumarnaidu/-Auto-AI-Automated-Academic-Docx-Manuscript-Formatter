# Abstract
This ACM benchmark checks section ordering, metadata presence, and bibliography stability.

# Introduction
Conference manuscripts need predictable formatting so authors can trust automated revisions.

## System Design
The benchmark feeds curated inputs into the formatter and compares the resulting DOCX structure to a golden summary.

## Findings
The current ACM template path keeps headings and bibliography entries stable enough for baseline testing.
