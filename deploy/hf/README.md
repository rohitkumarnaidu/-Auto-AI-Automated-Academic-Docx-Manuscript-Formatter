# HF Deployment Templates

This folder is only for Hugging Face Space deployment templates and copy guides.

- `grobid-service/`, `docling-service/`, `ocr-service/`, `docx-converter-service/`, `nougat-service/`, and `scibert-service/` contain docker Space templates.
- `COPY_TO_SPACES.md` maps all 12 production HF URLs and expected health paths.

These files are **not imported by backend runtime**; they are deployment assets only.
